How does this site work?

This is a static site. It's generated from a flat file database.
C code parses the text files and populates a sqlite db.
Bash scripts query the db and output html.
When I git push, it compiles my c tools and publishes the site with them.

Why did you do that?!

The footprint of the internet on the earth is growing.
We should use the simplest, lightest, and most long term tools that we control to tell our stories.
The way you deliver a story is as important as the content, and part of my story is taking the path less travelled.
